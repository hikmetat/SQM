StreamDispatcher::SQMDispatcherBase

Base class for as SAX-like event dispatcher for SQM input

SYNOPSIS

class MyClass is StreamDispatcher::SQMDispatcherBase {...}
my StreamBuffer $buffer .= new( :input($input)  ) ;

my MyClass $my-parser .=new( :stream-buffer( $buffer ) );
$my-parser.parse();

DESCRIPTION

This is the base class for as SAX-like event dispatcher for SQM handling. It extends StreamDispatcher::StreamDispatcherBase by providing a methods for the following events:

  • class-begin

  • attrib

  • class-end

  • array-begin

  • array-elem

  • array-end For each event a method of the form evp_the-event(%event-hash) exists which transform the hash input into 'name' and/or 'value' arguments and then call the event methods to be overridden by descendant classes of the form event-name(...). The method evp_closer yields to evp_class-end and evp_array-end according to the stack maintained by evp_class-begin and evp_class-end. Normally evp_* methods should not be overridden by descendant classes. All processing should be done under the event-name methods.
